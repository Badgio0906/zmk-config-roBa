# zmk-config-roBa

ZMK Firmware v0.3-branch（Zephyr 3.5）+ kumamuk-git PMW3610 ドライバ使用。

## キーマップ (Default Layer)

```
左手 (Left)                                             右手 (Right)
┌──────┬──────┬──────┬────────┬────────┐             ┌──────────┬───────┬──────┬──────┬──────┐
│  Q   │  L   │  U   │ Ctrl+C │ Ctrl+V │             │ Lang1/2  │  W/F  │  R   │  Y   │  P   │
├──────┼──────┼──────┼────────┼────────┼──────────┐  ├──────────┼───────┼──────┼──────┼──────┤
│ ,/.  │  I   │  A   │   O    │   -    │ W+Sft+S  │  │   Win    │   K   │  T   │  N   │  S   │  H
├──────┼──────┼──────┼────────┼────────┼──────────┤  ├──────────┼───────┼──────┼──────┼──────┤
│  Z   │  X   │  C   │   E    │   V    │ C+Sft+Esc│  │   ---    │   G   │  D   │  M   │  J   │  B
└──────┴──────┴──────┴────────┴────────┴──────────┘  └──────────┴───────┴──────┴──────┴──────┘
┌──────┬────────┬──────┬──────┬──────┬──────────┐  ┌──────────┬────────┐          ┌──────┐
│ ESC  │ Ctl/TB │ Alt  │ LSft │  BS  │  Sp/L1   │  │  Ent/L3  │ RSft   │          │ Del  │
└──────┴────────┴──────┴──────┴──────┴──────────┘  └──────────┴────────┘          └──────┘
```

- `Lang1/2` : タップ=英数、ダブルタップ=かな
- `W/F`     : タップ=W、ダブルタップ=F
- `,/.`     : タップ=,、ダブルタップ=.
- `Sp/L1`   : タップ=Space、ホールド=FUNCTION レイヤー
- `Ent/L3`  : タップ=Enter、ホールド=ARROW レイヤー
- `Ctl/TB`  : タップ=Tab、ホールド=Ctrl

## キーマップ (ARROW Layer) — `Ent/L3` ホールドで起動

```
左手 (Left)                                             右手 (Right)
┌──────┬──────────┬──────┬──────┬──────┐             ┌──────┬──────┬──────┬──────┬──────┐
│ ESC  │ C+S+TAB  │  ↑   │  *   │  (   │             │  ▽   │  ▽   │  ▽   │  ▽   │  ▽   │
├──────┼──────────┼──────┼──────┼──────┼──────────┐  ├──────┼──────┼──────┼──────┼──────┤
│ HOME │    ←     │  ↓   │  →   │ END  │ BT_CLR   │  │  ▽   │  ▽   │  ▽   │  ▽   │  ▽   │  ▽
├──────┼──────────┼──────┼──────┼──────┼──────────┤  ├──────┼──────┼──────┼──────┼──────┤
│ LSft │  W+S+←   │  ▽   │ W+S+→│  ▽   │    ▽     │  │  ▽   │  ▽   │  ▽   │  ▽   │  ▽   │  ▽
└──────┴──────────┴──────┴──────┴──────┴──────────┘  └──────┴──────┴──────┴──────┴──────┘
┌──────┬──────┬──────┬──────┬──────┬──────────┐  ┌──────────┬──────┐          ┌───────┐
│  ▽   │  ▽   │  ▽   │  ▽   │  ▽   │    ▽     │  │  (held)  │  ▽   │          │ RCtrl │
└──────┴──────┴──────┴──────┴──────┴──────────┘  └──────────┴──────┘          └───────┘
```

- `BT_CLR` : 現在のBLEプロファイルのペアリング情報を削除（BLE接続不能時の復旧用）
- `W+S+←/→` : Win+Shift+左右矢印（ウィンドウ移動）
